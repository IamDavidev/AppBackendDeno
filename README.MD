## How to run 🏃‍♂️

### you need

- [Deno](https://deno.land/)
- [NPM](https://www.npmjs.com/)

1. ⚡ Clone the repository

```bash
gh repo clone IamDavidev/DavTTer
```

2. ⚡ Install the dependencies

```bash
npm install dotenv-cli
```

3. ⚡ Add the environment variables

4. ⚡ run the project

```bash
deno task run::server
```
